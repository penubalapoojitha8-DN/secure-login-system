🔐 Secure Login System

📌 Description

A Python-based Secure Login System that allows users to register and log in securely using username and password authentication. The system validates credentials, hides passwords, and improves security using encryption/hash techniques.


---

🎯 Features

User Registration System

Secure Login Authentication

Password Hiding

Username & Password Validation

Prevents Unauthorized Access

Stores User Credentials Securely

Error Handling for Invalid Login

Beginner-Friendly Cyber Security Project



---

🛠️ Technologies Used

Python

File Handling / Database

Hashlib / bcrypt (for password hashing)

getpass module



---

▶️ How to Run

1. Download the project


2. Open folder in terminal


3. Run:
